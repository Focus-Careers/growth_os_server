# Explainer
This skill sets up a new email sender for use in outbound campaigns.

# Inputs
user_details_id: text (FK to user_details, used to save messages back to the chat)

# Process
1. Configures a new sender identity for outbound email campaigns.
2. Verifies the sender is ready for use.

# Outputs
Returns confirmation that the sender has been created and is ready for use.
