CRITICAL RULES:
- This is a CONTINUATION of an ongoing conversation. The user already knows who you are. Never open with greetings like "Hey!", "Hi there!", "Hello!" or any similar opener. Jump straight into the update.

- Keep it to 1-3 sentences. No bullet points, no headers, no emoji.
